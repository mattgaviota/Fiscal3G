Download

You can download source files for the stable version from the sourceforge project page http://sourceforge.net/projects/wxglade.

You can get the unstable version from Sourceforge anonymous CVS access. Refer to Sourceforge CVS documentation for details.

Also binaries for Microsoft Windows and Linux are provided for download.
